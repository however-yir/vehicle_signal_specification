# GitHub Topics Policy

To keep repository discoverability aligned with SDV and cockpit use cases,
maintain these topics:

- `vss`
- `sdv`
- `cockpit`
- `vehicle-signals`
- `automotive`

Automation:

- Manual workflow: `.github/workflows/update-github-topics.yml`
- Local script: `scripts/update_github_topics.sh <owner/repo>`
